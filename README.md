# Reinaldo Portifólio

Base limpa do novo portfólio em React + TypeScript + Vite, com arquitetura preparada para evolução de seções e páginas de case.

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React
- clsx + tailwind-merge

## Rodar localmente
```bash
npm install
npm run dev
```
